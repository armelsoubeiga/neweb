This course introduced students to shell programming on lunix. Over **18 hours** we covered:
* Program execution
* User groups
* Directory manipulation
* Files manipulation
* Files content manipulation
* Links
* Access permissions
* Redictions (>, >>)
* Variables
* Process identification
* Regular expressions
* Searching with grep
